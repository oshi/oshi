1   /**
2    * JNA-based native implementation of the OSHI API for JDK 8+.
3    * <p>
4    * This module uses <a href="https://github.com/java-native-access/jna">JNA</a> for native access and supports all OSHI
5    * platforms (Windows, macOS, Linux, Android, FreeBSD, OpenBSD, Solaris, AIX).
6    * <p>
7    * Usage:
8    *
9    * <pre>{@code
10   * SystemInfoProvider si = SystemInfoFactory.create();
11   * HardwareAbstractionLayer hal = si.getHardware();
12   * OperatingSystem os = si.getOperatingSystem();
13   * }</pre>
14   *
15   * Re-using the same {@link oshi.SystemInfo} instance optimizes caching and CPU performance; creating a new instance
16   * minimizes memory usage.
17   *
18   * @see oshi.SystemInfo
19   */
