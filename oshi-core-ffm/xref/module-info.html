1   /**
2    * FFM-based native implementation of the OSHI API for JDK 25+.
3    * <p>
4    * This module uses the Foreign Function and Memory (FFM) API for native access and currently supports Windows, macOS,
5    * and Linux. For broader platform support, use the JNA-based {@code com.github.oshi} module.
6    * <p>
7    * Usage:
8    *
9    * <pre>{@code
10   * SystemInfo si = new SystemInfo(); // oshi.ffm.SystemInfo
11   * HardwareAbstractionLayer hal = si.getHardware();
12   * OperatingSystem os = si.getOperatingSystem();
13   * }</pre>
14   *
15   * Re-using the same {@link oshi.ffm.SystemInfo} instance optimizes caching and CPU performance; creating a new instance
16   * minimizes memory usage.
17   *
18   * @see oshi.ffm.SystemInfo
19   */
