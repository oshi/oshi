1   /**
2    * FFM-based native implementation of the OSHI API for JDK 25+.
3    * <p>
4    * This module uses the Foreign Function and Memory (FFM) API for native access and currently supports Windows, macOS,
5    * Linux, FreeBSD, OpenBSD, and Solaris (illumos). For broader platform support (DragonFly BSD, NetBSD, AIX), use the
6    * JNA-based {@code com.github.oshi} module.
7    * <p>
8    * Usage:
9    *
10   * <pre>{@code
11   * SystemInfoProvider si = SystemInfoFactory.create();
12   * HardwareAbstractionLayer hal = si.getHardware();
13   * OperatingSystem os = si.getOperatingSystem();
14   * }</pre>
15   *
16   * Re-using the same {@link oshi.ffm.SystemInfo} instance optimizes caching and CPU performance; creating a new instance
17   * minimizes memory usage.
18   *
19   * @see oshi.ffm.SystemInfo
20   */
